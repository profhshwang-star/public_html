1. 프롬프트 기본구조

Prompt Structure / 프롬프트 구조

기본 구조
비디오 형식 + 카메라 무브먼트 + 주제/액션 + 환경/배경 + 시각적 스타일 + 조명/색상 + 시간적 요소 + 특수효과 + 캐릭터 + 브랜딩 + 플랫폼 + 품질 파라미터
고급 구조
[목적/형식] + [기본 컨셉] + [시각적 스타일] + [카메라워크] + [주요 액션] + [환경/배경] + [캐릭터/피사체] + [조명/색상] + [시간적 처리] + [특수 효과] + [무드/감정] + [기술적 사양] + [플랫폼 최적화]
확장된 사용 가이드
  • 각 카테고리에서 2-3개의 핵심 키워드 선택 권장
  • 상충되는 요소는 우선순위에 따라 배치
  • 네거티브 프롬프트 활용으로 원하지 않는 요소 제외
  • 가중치 조절로 특정 요소 강조 가능 (예: strong lighting:1.5)
  • 조건부 프롬프트로 복잡한 시나리오 구성
Negative Prompts / 네거티브 프롬프트

원하지 않는 요소들을 명시적으로 제외:

avoid shaky camera
흔들린 카메라 제외
no blur
흐림 없음
avoid overexposure
과다 노출 방지
no distortion
왜곡 없음
avoid harsh shadows
강한 그림자 방지
